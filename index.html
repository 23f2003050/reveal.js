Code Sample: Portfolio Return

Python script for calculating weighted portfolio returns.


# Define asset weights and their returns
weights = [0.5, 0.3, 0.2]  # 50% stock, 30% bonds, 20% cash
returns = [0.12, 0.05, 0.01] # 12% stock, 5% bonds, 1% cash

# Calculate weighted return
portfolio_return = sum(w * r for w, r in zip(weights, returns))

print(f"Expected Portfolio Return: {portfolio_return:.2%}")

Financial Modeling: CAPM

We use the Capital Asset Pricing Model (CAPM) to calculate the expected return on an asset:

$$ E(R_i) = R_f + \beta_i (E(R_m) - R_f) $$

Where:

  • \( E(R_i) \) = Expected return of the investment
  • \( R_f \) = Risk-free rate
  • \( \beta_i \) = Beta of the investment
  • \( E(R_m) \) = Expected return of the market
This is a fundamental formula we use for asset valuation. The key variables are the risk-free rate, beta, and the expected market return. It helps us determine if an asset is fairly priced.
